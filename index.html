Image names

Enter the file name of your images, leaving off any size suffix that a tool like Shrinkray may have added. For example, if your file name is image1_1800.jpg, just enter image1. Separate the names by a comma. You should have as many image names as you have number of slides.

Image sizes

Enter the sizes of your images here, largest to smallest. Again, separate the sizes by a comma. Also, only the size is needed, not the unit of measurement.
